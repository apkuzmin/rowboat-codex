## What is Rowboat Open?

`Rowboat Open` is an updated version of `Rowboat` focused on integrating `ChatGPT OAuth` as an AI provider in a practical local-first setup.

This fork keeps the core `Rowboat` model of local memory, notes, and context-aware assistance, but reframes the project around the implementation goals of this repository.

## What Is Different In This Version

- `ChatGPT OAuth` is integrated as one of the AI provider options
- the repository description is aligned with this fork rather than the upstream product marketing
- the project still supports local memory, Markdown-based context, and MCP-based extensions

## What The Project Does

The project uses local context to help with:
- meeting prep
- summaries and notes
- document and artifact generation
- integration with email, calendar, and local knowledge storage

Under the hood, the project keeps a local Markdown-based memory layer that can be inspected and edited directly.

## Model Providers

This version supports:
- **Local models** via `Ollama` or `LM Studio`
- **Hosted models** via external providers
- **ChatGPT OAuth** as an integrated AI provider

## Integrations And Extensions

The project can be extended through **MCP** and other local integrations. Typical integrations include email, calendar, search, voice services, and custom internal tools.
